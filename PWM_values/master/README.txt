Naming conventions:

- CL -> B; CALCL.TXT -> CALB.TXT and PWMCL.TXT -> PWMB.TXT
- CR -> C; CALCR.TXT -> CALC.TXT and PWMCR.TXT -> PWMC.TXT

CALx.TXT includes:
    - offset        -> interger
    - phaseshift    -> integer
    - P_pos         -> integer
    - I_pos         -> double
    - D_pos         -> double
    - P_speed       -> double
    - I_speed       -> double
    - D_speed       -> double
    - P             -> integer optional
    - I             -> double  optional
    - D             -> double  optional
